### 2. install Node.js

Download [Node.js v6.9.1](https://nodejs.org/dist/v6.9.1/node-v6.9.1-x64.msi) and install.
v6.0.0 or more is required.

Command Prompt (Terminal)  Node
```sh
$ node -v
v6.9.1
```

### 3. Install Node packages

```sh
$ npm install gulp -g
$ npm install
```

### 4. Run Gulp

```sh
$ gulp
```
